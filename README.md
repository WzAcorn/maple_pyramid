# maple_pyramid
자바응용프로그램 최종 과제

# 프로젝트 소개.
몬스터 피라미드는 메이플스토리 에서 2019년 4월 25일에 출시한 미니게임으로 간단한 조작방식에 비해 머리를 많이 쓰게 되어, 
미니게임엔 박한 평가를 자주 주던 메이플 게임 유저들에게도 호평을 받았던 게임입니다. 저희는 이 게임을 최대한 똑같이 구현하는 것을 목표로 개발을 진행하였습니다.
